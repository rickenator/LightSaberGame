# Vision

Arcblade is a fast, responsive, single-player Android energy-blade action game
optimized for short, high-focus sessions.

## Session goal

- Typical play session: 5-10 minutes
- Immediate action with low friction
- Strong feel from motion, sound, and feedback

## Product boundaries

Arcblade is intentionally:

- Not an MMO
- Not a 3D AAA production
- Not competitive multiplayer

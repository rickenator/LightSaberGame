# ADR 0004: Product name and namespace

- Status: Accepted
- Date: 2026-05-25

## Context

The project requires neutral product branding and consistent Android namespace
identifiers.

## Decision

Adopt product name `Arcblade` and namespace `com.aniviza.arcblade`.

## Consequences

- Human-readable documentation and product references use Arcblade
- Java package and Android namespace are standardized
- GitHub repository slug remains `LightSaberGame` for historical continuity

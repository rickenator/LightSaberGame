# ADR 0003: Use Apache-2.0 license

- Status: Accepted
- Date: 2026-05-25

## Context

The project should remain permissive and compatible with common open-source
library usage patterns.

## Decision

License Arcblade under Apache License 2.0.

## Consequences

- Broad compatibility for code sharing and integration
- Clear patent and redistribution terms
- Attribution and notice requirements apply

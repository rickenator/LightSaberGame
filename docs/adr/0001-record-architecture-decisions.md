# ADR 0001: Record architecture decisions

- Status: Accepted
- Date: 2026-05-25

## Context

Long-running projects benefit from explicit records of why key decisions were
made.

## Decision

Adopt lightweight ADRs in `docs/adr/` using a MADR-lite structure.

## Consequences

- Better continuity across contributors
- Easier technical onboarding
- Clear rationale for revisiting prior choices
